# Can You Get 10
Like the game [Can You Get 11](https://itunes.apple.com/us/app/can-you-get-11-simple-fun/id1044945491?mt=8) by Shimizu Naoki but easier.

## Requirements

- npm 
- gulp 
- jspm

## Installation

1. Install dependencies from the project directory: `$ npm install`

2. Run Gulp: `$ gulp`
